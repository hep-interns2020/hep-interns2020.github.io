introduction to web development

what is web development?

Web devlopment is the building, creating, and maintenance of websites on the internet; it's the work that happens behind the scenes that enable website functionality, makes the website look nice and for it to perform smoothly for the user. It can range from developing a simple page of texts to more complex website-based internet applications, electronic businesses, and social networking services.


what did the web development cohort do?

Interns led by an undergrad manager, work together to learn and develop the skills about different aspects of web development including web programming and web design through different coding languages. The interns focus on 3 main coding languages (HTML, CSS, and Javascript) to assist them in building this awesome website! Theh interns were also able to explore the different parts of coding to create ideas for the website.
